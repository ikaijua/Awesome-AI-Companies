# Contributing to Awesome AI Companies

Thank you for your interest in contributing! We welcome all contributions that help make this a comprehensive resource for the AI industry chain.

## How to Contribute

### 1. Adding a New Company
To add a new company, please ensure it fits into one of the existing categories. Use the following format:

| Name | Description | Links | Headquarters |
| --- | --- | --- | --- |
| Company Name | A brief description of what the company does and its significance in the AI industry. | [URL](https://example.com) | Country/Region |

### 2. Suggesting a New Category
If you think a major part of the AI industry chain is missing, feel free to open an issue to discuss it.

### 3. Improving Existing Content
If you find outdated information or broken links, please open a pull request with the fixes.

## Guidelines
- **Be Concise:** Keep descriptions short and informative.
- **Provide Links:** Always include the official website of the company.
- **Bilingual Sync:** If you add a company to `README.md`, please try to add it to `README-CN.md` as well (or at least let us know in the PR).

## Submission Process
1. Fork the repository.
2. Create a new branch for your changes.
3. Commit your changes.
4. Push to your fork and submit a pull request.

We look forward to your contributions!
