# 贡献指南

感谢您对本项目感兴趣！我们欢迎任何有助于使本项目成为 AI 产业链全面资源库的贡献。

## 如何贡献

### 1. 添加新公司
添加新公司时，请确保其符合现有分类。请使用以下格式：

| 名称 | 说明 | 链接 | 总部 |
| --- | --- | --- | --- |
| 公司名称 | 简要说明公司业务及其在 AI 行业中的地位。 | [URL](https://example.com) | 国家/地区 |

### 2. 建议新分类
如果您认为遗漏了 AI 产业链的某个重要环节，欢迎通过 Issue 进行讨论。

### 3. 完善现有内容
如果您发现信息过时或链接失效，请提交 Pull Request 进行修复。

## 指南
- **简洁明了：** 描述应简短且信息丰富。
- **提供链接：** 务必包含公司的官方网站。
- **双语同步：** 如果您在 `README.md` 中添加了公司，请尽量也在 `README-CN.md` 中添加（或者在 PR 中告知我们）。

## 提交过程
1. Fork 本仓库。
2. 为您的更改创建一个新分支。
3. 提交更改。
4. 推送到您的 Fork 并提交 Pull Request。

期待您的贡献！
